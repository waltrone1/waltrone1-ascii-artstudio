# Changelog

All notable changes to **waltrone1 ASCII ArtStudio** will be documented in this file.

This project is part of the **WALTRONE Community Tools** collection.

---

## [1.0.0.0] - 2026-05-17

### Added

- Initial public release of waltrone1 ASCII ArtStudio
- Image to ASCII conversion for common image formats
- Support for JPG, PNG, BMP, GIF and WEBP images
- Multiple ASCII character presets
- Custom character set support
- Optional custom WALTRONE / handle based character output
- Original image color rendering
- Retro palette rendering
- Matrix, amber, cyber, arcade, neon and CRT-inspired palette styles
- Retro CRT, glow, glitch, terminal and arcade-style effects
- Adjustable ASCII width
- Adjustable font size, line height and letter spacing
- Dark-pixel transparency and empty-space handling
- Live browser preview for HTML output
- Automatic preview refresh while editing settings
- HTML export
- Standalone HTML export
- Embed HTML export
- SVG / vector export
- PNG export
- Transparent PNG export option
- Terminal frame export mode
- Clean art-only export mode
- Branded export mode
- Retro arcade motif support
- Copyable ASCII output in HTML exports
- Windows desktop interface
- Application icon
- PyInstaller / py2exe build files for creating a Windows executable
- GitHub repository structure with README, LICENSE, CHANGELOG, screenshots and documentation folders
- README screenshot reference for `screenshots/ascii-artstudio-main.png`

### Notes

- waltrone1 ASCII ArtStudio is a creative image conversion and export tool.
- The application does not automatically upload, publish or modify original image files.
- Very large images or high detail settings may take longer to process.
- HTML output may render slightly differently depending on browser and font rendering.
- Users should only process and publish images they are allowed to use.
